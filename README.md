\# Student Management System App



\## Tech Stack

\- Spring Boot (Backend)

\- Angular (Frontend)

\- H2 Database



\## Project Structure

backend/ - Spring Boot app  

frontend/ - Angular app  



\## Run Backend

cd backend  

mvn spring-boot:run  



\## Run Frontend

cd frontend  

npm install  

ng serve  



\## URLs

Frontend: http://localhost:4200  

Backend: http://localhost:8080  



\## Features

\- CRUD Students \& Courses

\- Validation \& Exception Handling

\- Reactive Forms

\- Routing

\- Confirmation on Delete
